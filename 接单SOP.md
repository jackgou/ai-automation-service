# 📋 AI自动化服务 - 接单SOP

## 一、接单流程

```
客户咨询 → 需求沟通 → 报价确认 → 签约付款 → 开发交付 → 验收完成
```

### 1. 客户咨询（5分钟内响应）

**开场白模板：**
```
您好！我是AI自动化服务专员。
请问您需要什么类型的自动化服务？
我们提供电商/自媒体/企业等多种场景的AI自动化解决方案。
```

### 2. 需求沟通（10-15分钟）

**必问问题：**
- [ ] 您的行业/领域是什么？
- [ ] 具体想自动化什么流程？
- [ ] 目前手动操作的痛点是什么？
- [ ] 期望的交付时间？
- [ ] 预算范围？

### 3. 报价确认

**报价公式：**
```
基础价 = 场景复杂度 × 基础单价
- 简单场景（标准模板）：¥500
- 中等场景（轻度定制）：¥1000-1500
- 复杂场景（深度定制）：¥2000-3000

附加费用：
+ 加急（24h内）：+30%
+ 源码交付：+20%
+ 1对1培训：+300/小时
```

**报价模板：**
```
您好，根据您的需求，报价如下：

📋 服务内容：[具体服务]
💰 服务费用：¥[价格]
📅 交付周期：[X]个工作日
📦 交付清单：
   - [清单项1]
   - [清单项2]
   - [清单项3]

付款方式：先付50%，验收后付50%
```

### 4. 签约付款

- 提供收款码/转账信息
- 确认付款后开始开发
- 建立专属沟通群

### 5. 开发交付

- 每日进度同步
- 交付前内部测试
- 提交验收

### 6. 验收完成

- 客户验收确认
- 收取尾款
- 发送满意度调查

## 三、异议处理

**异议1："太贵了"**
→ 回应："我们是专业团队，交付质量有保障。而且相比您手动操作的时间成本，其实非常划算。要不我先做个Demo给您看看效果？"

**异议2："我自己能学"**
→ 回应："当然可以！但时间也是成本。我们已经积累了大量行业经验，可以帮您快速落地，避免踩坑。"

**异议3："能便宜点吗"**
→ 回应："老客户续费9折，推荐新客户返现10%。长期合作更划算。"
